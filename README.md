# AGATEM Next.js Landing Page

## Run locally

```bash
npm install
npm run dev
```

## Build for production

```bash
npm run build
npm run start
```

## Included
- `app/page.tsx` live-ready landing page
- `app/layout.tsx` app shell
- `app/globals.css` styling
- `public/` product images

## Notes
- The lead form opens WhatsApp with the captured buyer details
- Update the WhatsApp number in `app/page.tsx` if needed
- If you want analytics or pixel events, add them in `app/layout.tsx` or `app/page.tsx`
